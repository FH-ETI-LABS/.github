# .github

This repository contains the organization profile and community health files for FH ETI Labs.

## Contents

- `profile/README.md` - Organization profile displayed on https://github.com/FH-ETI-LABS

## About Community Health Files

Files in this repository can be used as defaults across all repositories in the organization, including:
- CODE_OF_CONDUCT.md
- CONTRIBUTING.md
- FUNDING.yml
- SECURITY.md
- SUPPORT.md

Learn more about [creating a default community health file](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
